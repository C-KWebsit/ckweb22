Data Types

Definition of Javascript Data Types:

Javascript Strings:
Strings are a series of characters that are written with quotes, you can use single or double
For example: let hello = 'Hello Everyone!';
Javascript Numbers:
Numbers are written with or without decimals
For example: let x = 34; let y = 35.35;
Javascript Booleans:
This data type returns either of the two values: true or false...Boolean is used as a function to get the value of a variable, object, conditions, expressions, etc. in terms of true or false
Javascript Arrays:
An array is a variable that can hold more than one value at one time under one single name.
const array = [item1, item2, item 3, item4]
Javascript Objects:
Objects are containers for named values called properties and methods.
